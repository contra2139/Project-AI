## 🔐 Authentication
Endpoints for managing user access and session security.

### POST `/auth/login`
Authenticates a user and returns JWT tokens.
- **Request Body:** `{ "username": "admin", "password": "..." }`
- **Response (200):** `{ "success": true, "data": { "access_token": "...", "token_type": "bearer", "expires_in": 3600 } }`
- **Security:** Rate limited (5 attempts/min).

### POST `/auth/refresh`
Refreshes the access token using a refresh token (stored in cookies).
- **Response (200):** New access token.

### POST `/auth/logout`
Revokes the refresh token and clears session data.

## 📊 Strategy & Configuration
Endpoints for managing symbols and bot settings.

### GET `/symbols/`
Lists all tracked symbols and their configurations.
- **Headers:** `Authorization: Bearer {token}`
- **Response:** List of symbol metadata.

### PATCH `/settings/mode`
Updates the global bot execution mode (e.g., Live, Paper, Scan-Only).
- **Headers:** `Authorization: Bearer {token}`
- **Request Body:** `{ "mode": "paper" }`
